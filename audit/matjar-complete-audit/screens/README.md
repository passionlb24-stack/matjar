# لقطات الشاشة — Screens

## لماذا لا توجد ملفّات صور هنا

تعذّر حفظ لقطات شاشة كملفّات ضمن هذا التدقيق، للأسباب التالية:

1. **الأدوات:** متصفّح البيئة (in-app browser) يُرجع اللقطة كصورة داخل المحادثة فقط، ولا يحفظها كملفّ على القرص داخل المستودع؛ كما أنّ جزء العرض (Browser pane) لم يكن مُركّباً (not compositing) وقت التدقيق فتعذّر الالتقاط.
2. **الصفحات المحميّة:** لوحات التاجر و**Super Admin** محميّة بتسجيل دخول (Supabase Auth + RLS + `admin_can()`)، والمتصفّح الآليّ في هذه الجلسة غير مسجّل بحساب المالك، فلا يمكن الوصول إليها لالتقاطها. هذا **قيد بيئة تدقيق**، لا خلل في المنتج.

## بديل: التدقيق البصريّ تمّ من الكود مباشرةً

بدل الاعتماد على لقطات، دُقّقت الواجهات من **الكود الفعليّ** (المكوّنات، أصناف Tailwind، `globals.css`، منطق الاستجابة والـRTL) — راجع:
- `03_UX_UI_AUDIT.md` — تدقيق صفحة-صفحة وعنصر-عنصر.
- `05_RESPONSIVE_DEVICE_AUDIT.md` — نقاط الكسر عبر الأجهزة.
- `09_ACCESSIBILITY_AUDIT.md` — التباين والوصول.

## لالتقاط اللقطات لاحقاً (توصية)
شغّل محلّياً `npm run dev`، وسجّل دخولاً كـ **super_admin** وكـ **تاجر**، والتقط:
الرئيسيّة (سطح مكتب + موبايل)، تسجيل التاجر، رحلة العميل، لوحة التاجر، لوحة Super Admin، صفحة المتجر، صفحة المنتج، صفحة الخدمة، مركز الأعمال، الأكاديميّة، رجال الأعمال، وأي صفحة معطوبة.
